# nyantracker player

A web-based player for chiptune music modules, built with WebAssembly and TypeScript. It supports various module formats and provides a user-friendly interface for playback and navigation. Using [DrSnuggles's chiptune3](https://github.com/DrSnuggles/chiptune) and libopenmpt.
